# Accounts

## Hướng Dẫn Sử Dụng Lệnh `accounts`

Lệnh `accounts` cho phép bạn xem thông tin các tài khoản đã liên kết của mình

### Cách Sử Dụng

```bash
/accounts
```

### Tùy Chọn (Options)

- Không có tùy chọn

### Ví Dụ

1. **Sử Dụng Cơ Bản:**

  ```bash
  /accounts
  ```
