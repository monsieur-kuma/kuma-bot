# Checkin Games

## Hướng dẫn cho lệnh `checkin_games`

Lệnh `checkin_games` cho phép bạn checkin tất cả các trò chơi bạn đã liên kết với tài khoản

### Cách sử dụng

```bash
/checkin_games
```

### Tùy chọn

- Không có tùy chọn

### Ví dụ

1. **Cách sử dụng cơ bản:**

  ```bash
  /checkin_games
  ```
