# Redeem Code

## Hướng dẫn cho lệnh `redeem_code`

Lệnh `redeem_code` hướng dẫn bạn cách đổi mã trong trò chơi

### Cách sử dụng

```bash
/redeem_code [game] [code]
```

### Tùy chọn

- `game`: Trò chơi bạn muốn đổi mã
- `code`: Mã phần thưởng

### Ví dụ

1. **Cách sử dụng cơ bản:**

  ```bash
  /redeem_code [game: hsr] [code: STARRAILGIFT]
  ```
