# Link Hoyolab

## Tutorial for `link_hoylab` Command

The `link_hoylab` command help you link with hoyolab account

### Usage

```bash
/link_hoylab [games]
```

### Options

- `games`: game you want to link. You can select multiple game

### Examples

1. **Basic Usage:**

  ```bash
  /link_hoylab [game-you-want-to-link]
  ```
