# Guide Link

## Hướng Dẫn Sử Dụng Lệnh `guide_link`

Lệnh `guide_link` hiển thị hướng dẫn liên kết tài khoản hoyolab

### Cách Sử Dụng

```bash
/guide_link
```

### Tùy Chọn

- Không có tùy chọn

### Ví Dụ

1. **Cách Sử Dụng Cơ Bản:**

  ```bash
  /guide_link
  ```
